# Evaluation

- **Objective**: Total energy cost ($), % cost reduction vs uncontrolled (charge-asap), peak load (kW).
- **Feasibility/service**: Total unmet energy, % EVs fully served, constraint violation count.
- **Faithfulness**: Verify numeric claims in explanations match computed schedule (cost, savings, peak, unmet).
- **Stretch**: Sweep λ; cost–peak tradeoff.
- Compare baseline vs agent over 5+ days, varying congestion; fixed TOU vector per experiment.
